## DMIT 是什么，为什么在 VPS 圈口碑不错

简单说：DMIT 是 2018 年在美国纽约注册的 VPS 服务商，从创立起就走"自建机房 + 自有线路"这条路，不转卖上游资源。这一点在 VPS 行业里挺少见。多数便宜 VPS 商家的本质是中间商，线路质量跟着上游走，上游抖一抖你的服务器就跟着抖。DMIT 自己掌控带宽，所以对中国方向的线路稳定性有更强的话语权。

目前有四个机房：**美国洛杉矶（LAX）**、**美国圣何塞（SJC）**、**中国香港（HKG）**、**日本东京（TYO）**。

硬件方面，全系标配 AMD EPYC 处理器（9004 系或 9005 系，具体看套餐），企业级 SSD，KVM 虚拟化，默认给 1 个 IPv4 + 1 个 IPv6 /64。支持支付宝、微信、PayPal 付款，有中文客服。

## 先搞清楚三条线路的实际区别

DMIT 最让人困惑的就是产品线命名。Premium / Eyeball / Tier 1，这三个词代表网络档次，不是配置档次。同样 1 核 1G 的套餐，Premium 和 Tier 1 的网络体验差距可以很大。

**Premium（Pro 系列）**

三网 CN2 GIA，电信联通移动去回程全走 CN2 GIA 高端路由。这是大陆访问延迟最低的方案，晚高峰表现也能撑住。洛杉矶节点实测晚高峰大陆方向延迟基本维持在 150ms 上下，香港更快，通常在 20–50ms 区间。代价是价格。

**Eyeball（EB 系列）**

洛杉矶版走电信联通 AS9929、移动 CMIN2 回程；香港版移动双程 CMI，电信联通绕日本 NTT 回程直连。比 Premium 便宜，对联通移动用户来说性价比更高，但电信用户用下来不如 Pro 系列稳。

**Tier 1（T1 系列）**

纯国际线路，没有针对大陆的特殊优化，但价格低，年付最便宜的只要 $36.9。适合面向海外的业务，或者用来搭中转节点、CDN 边缘。流量用完后不关机，直接降速继续跑，这对大流量场景来说挺友好。

## 几个常见疑问

**IP 被墙了怎么办？**

每 15 天可以免费申请更换一次，前提是 IP 确实被墙了。其他情况下换 IP 收费 $5。这个策略相对透明，比那种"工单催了半天最后告诉你不保证 IP 可用"的商家好处理多了。

**默认是密码登录还是密钥登录？**

DMIT 默认用 SSH 密钥登录，安全性更高。如果不熟悉这套流程，官网有对应的中文教程，操作不复杂。

**套餐可以升级降级吗？**

支持升级，不支持降级。第一次买的话建议保守一点，确认网络表现满意再升配或者换成年付。

**流量超了会停机吗？**

T1 系列和部分 Eyeball 套餐超量后不停机，降速继续用。Pro 系列的超量机制不完全一样，建议下单前在官网确认具体套餐的规则。

**退款政策是怎样的？**

3 天内且流量用量不超 30GB，可以申请全额退款（扣支付网关手续费）；30 天内可以申请部分退款，按剩余时间比例折算。这个政策对"先买来测一测"的用户来说还算友好。
